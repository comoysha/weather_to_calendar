# 需求
写一个脚本，能查询天气接口，解析返回中的信息，存到日历中
## 高德天气接口
### 请求
curl --location --request GET 'https://restapi.amap.com/v3/weather/weatherInfo?Key=23c87799138b09c5b8de14d81c4d7eea&city=330110&'

### 返回
```json
{
    "status": "1",
    "count": "1",
    "info": "OK",
    "infocode": "10000",
    "lives": [

            "city": "余杭区",

            "weather": "阴",
```

## 参考这种创建日历日程的AppleScript命令示例
```
local apple_script="
tell application \"Calendar\"
    set targetCalendar to first calendar whose name is \"$calendar_name\"
    
    set startDate to (date \"$start_date\")
    set endDate to (date \"$end_date\")
    
    tell targetCalendar
        make new event with properties {summary:\"$event_title\", start date:startDate, end date:endDate, description:\"从${target_date}到今天的年龄计算\"}
    end tell
end tell
"
```

## 存到日历的日程
### 标题
示例
```
32°C，阴，65%
```
解释：
用返回中的temperature、weather、humidity，拼接成这样的格式（要带上温度符号°C和湿度的%）
### 日程时间
start_date=早上7点
end_date=早上7点
